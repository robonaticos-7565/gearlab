Módulos do Gear Up

Gear Up Modules

Módulos de Gear Up

Nossos módulos foram desenvolvidos para proporcionar uma experiência de aprendizado completa e envolvente. Cada projeto está alinhado com a Base Nacional Comum Curricular (BNCC) e pode ser aplicado de forma autônoma pelos professores.

Our modules were developed to provide a complete and engaging learning experience. Each project is aligned with the National Common Curricular Base (BNCC) and can be applied autonomously by teachers.

Nuestros módulos fueron desarrollados para proporcionar una experiencia de aprendizaje completa y atractiva. Cada proyecto está alineado con la Base Nacional Común Curricular (BNCC) y puede ser aplicado de forma autónoma por los profesores.

Module Content

Contenido de los Módulos

Módulo 1: Gear Up Básico

Module 1: Basic Gear Up

Módulo 1: Gear Up Básico

Projetos voltados para Ensino Fundamental 2. Primeiro módulo criado. Robô de luz como principal projeto.

Projects aimed at Elementary School 2. First module created. Light robot as the main project.

Proyectos dirigidos a Educación Básica 2. Primer módulo creado. Robot de luz como proyecto principal.

Módulo 2: Gear Up Fórmula Maker

Module 2: Gear Up Formula Maker

Módulo 2: Gear Up Fórmula Maker

Projetos sem a utilização de componentes eletrônicos, apenas voltado para engenharia. Criado durante a pandemia para auxiliar professores na Educação a Distância.

Projects without the use of electronic components, focused only on engineering. Created during the pandemic to assist teachers in Distance Education.

Proyectos sin el uso de componentes electrónicos, enfocados solo en ingeniería. Creado durante la pandemia para ayudar a los profesores en la Educación a Distancia.

Módulo 3: Gear Up High Tech

Module 3: Gear Up High Tech

Módulo 3: Gear Up High Tech

Projetos maker utilizando programação avançada. Voltado para estudantes do Ensino Médio. Robô de sumô como principal projeto.

Maker projects using advanced programming. Aimed at High School students. Sumo robot as the main project.

Proyectos maker utilizando programación avanzada. Dirigido a estudiantes de Educación Secundaria. Robot de sumo como proyecto principal.

Módulo 4: Gear Up Artes

Module 4: Gear Up Arts

Módulo 4: Gear Up Artes

Mescla artes com tecnologia e inovação. Projetos que incentivam a criatividade em alunos de sua maioria no Ensino Médio/ Ensino Fundamental.

Blends arts with technology and innovation. Projects that encourage creativity in students, mostly in High School/Elementary School.

Mezcla artes con tecnología e innovación. Proyectos que fomentan la creatividad en estudiantes, principalmente de Educación Secundaria/Básica.

Módulo 5: Gear Up Lego

Module 5: Gear Up Lego

Módulo 5: Gear Up Lego

Criado para atender o hospital de crianças com câncer (GRAACC). 100% esterilizável, utilizando de materiais LEGO para a educação de robótica e tecnologia.

Created to serve the children's cancer hospital (GRAACC). 100% sterilizable, using LEGO materials for robotics and technology education.

Creado para atender el hospital de niños con cáncer (GRAACC). 100% esterilizable, utilizando materiales LEGO para la educación de robótica y tecnología.
